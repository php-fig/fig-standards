### 2.6. Data Types

When casting to a boolean or an integer, use the abbreviated form of the data
type name (e.g. `bool` and `int`).

```
$b = (bool)$a;
$y = (int)$x;
```

When casting to or testing for a float or a double, use the `float` data type /
functions.

```
$y = (float)$x;
$z = is_float($y);
```
